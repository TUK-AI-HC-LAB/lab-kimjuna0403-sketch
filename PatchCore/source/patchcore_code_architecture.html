파일 구조 요약

  • • bin/run_patchcore.py — CLI 진입점 (click)
  • • src/patchcore/patchcore.py — 핵심 클래스
  • • src/patchcore/common.py — 모든 하위 모듈
  • • src/patchcore/sampler.py — Coreset 로직
  • • src/patchcore/backbones.py — 백본 로더
  • • src/patchcore/datasets/ — MVTec, MTD 등

핵심 클래스 관계

  • • PatchCore.forward_modules = ModuleDict
  • • feature_aggregator → forward hook
  • • PatchMaker → unfold (이웃 집계)
  • • featuresampler → .run() → coreset
  • • anomaly_scorer → FaissNN.run()
  • • fit() / predict() 두 메서드로 구성

저장 & 실행 방식

  • • FAISS 인덱스 → .faiss 파일로 저장
  • • 하이퍼파라미터 → .pkl 파일로 저장
  • • GPU/CPU 모두 지원 (--faiss_on_gpu)
  • • 백프로파게이션 없음 (학습 없음)
  • • W&B 온라인 로깅 지원 (--log_online)
