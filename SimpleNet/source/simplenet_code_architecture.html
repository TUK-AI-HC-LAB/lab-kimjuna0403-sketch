파일 구조 (루트 플랫)

  • • main.py — CLI 진입점 (click)
  • • simplenet.py — SimpleNet + Projection + Discriminator
  • • common.py — 피처 추출 유틸 (PatchCore 공유)
  • • backbones.py + resnet.py — 백본 로더
  • • metrics.py / utils.py — 평가 & 유틸
  • • datasets/mvtec.py — 데이터셋

PatchCore와의 구조 비교

  • • 공유: forward_modules, common.py 구조
  • • 제거: FAISS kNN, Coreset sampler
  • • 추가: Projection (Feature Adaptor)
  • • 추가: Discriminator (MLP 분류기)
  • • 추가: Gaussian noise 합성 (train-only)
  • • 추가: backprop 학습 (Projection + DSC)

학습 방식

  • • meta_epochs=160 / batch_size=4
  • • AdamW (backbone, lr=1e-4)
  • • Adam (Discriminator, lr=2e-4)
  • • CosineAnnealingLR 스케줄러
  • • Truncated L1 Loss (BCE 아님)
  • • 77 FPS 추론 (FAISS 없어서 빠름)
