파일 구조 (루트 플랫)

  • • main.py — argparse 진입점 + 학습 루프
  • • test.py — evaluation / visualization / test
  • • resnet.py — Teacher Encoder 정의
  • • de_resnet.py — Student Decoder + OCBE(bn)
  • • dataset.py — MVTecDataset
  • • (별도 src/ 폴더 없음 — 완전 플랫 구조)

3-component 핵심 구조

  • • encoder (resnet.py) — 가중치 고정, 참조 기준
  • • bn (OCBE) — MFF + OCE, de_resnet.py 내 정의
  • • decoder (de_resnet.py) — encoder 대칭 역구조
  • • Adam(decoder + bn) 함께 학습
  • • 학습 후: encoder vs decoder 피처 차이로 탐지

3개 논문 비교 요약

  • • PatchCore: FAISS kNN + coreset, 학습 없음
  • • SimpleNet: Discriminator + Gaussian noise, 학습
  • • RD4AD: Teacher-Student KD, cosine loss 학습
  • • 공통: WideResNet50 backbone, MVTec AD
  • • RD4AD만 역방향 distillation (high→low)
